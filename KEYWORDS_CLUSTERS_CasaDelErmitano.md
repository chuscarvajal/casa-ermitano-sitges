## PASO 1 — LIMPIEZA (qué se eliminó y por qué)

El CSV venía contaminado. Se eliminó:

- **Vertical equivocada (ruido masivo):** todo "cursos/clases de inglés, francés, italiano, matemáticas, finanzas, oposiciones, contabilidad, reiki…". No tiene nada que ver con el negocio (export de cuenta mezclada).
- **Informativo / DIY (no transaccional):** todos los "**cómo** educar / adiestrar / entrenar / criar a un [perro/cachorro/raza/edad]", "cómo hacer que mi cachorro orine…", "para qué sirve la obediencia". El usuario quiere **hacerlo él**, no contratar. → fuera (sirven, como mucho, para blog futuro, no para páginas de venta).
- **Gratis / PDF / producto:** "adiestramiento canino pdf", "entrenamiento de perros gratis", "clicker para perros", libros ("turid rugaas", "karen pryor", "psicología del aprendizaje y adiestramiento del perro").
- **Marcas y nombres propios de competencia:** "el adiestrador de perros sant cugat", "los guardianes", "petit melic", "enric palau adiestrador", "leonardo paonessa", "modest dog", etc.
- **Fuera de zona:** Sant Cugat, Gran Canaria, Ciudad Real, Santo Domingo, Santa Fe, zona norte/oeste… (LATAM y otras provincias). La zona es el Garraf.
- **Otras especies / fuera de servicio declarado:** adiestrar gato, loros, conejos, periquito, agapornis; "ser adiestrador canino", "adiestramiento de personas" (formarse como adiestrador); perros de caza / agility / IGP / deportivo (nichos que Eduardo no mencionó).
- **Lingüístico puro:** "obedecer / obediencia / obediente en inglés / hebreo / definir / sinónimo".

> Resultado: de ~1.000 filas quedan ~120 keywords realmente relevantes y transaccionales, agrupadas abajo.
